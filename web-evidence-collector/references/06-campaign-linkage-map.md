# Campaign Linkage Map

Use this file to show how evidence connects across channels. This is collection structure, not strategic synthesis.

## Purpose

Downstream skills need to know whether video, offline, marketing, PR, visual, and social evidence are isolated examples or part of the same campaign chain.

## Linkage Signals

Treat evidence as related when one or more are true:

- Same campaign name, event name, hashtag, slogan, repeated message, or product launch.
- Official source explicitly connects the assets.
- Same landing page links to the video, event, promotion, or PR release.
- Same release window and repeated campaign language across channels.
- Media coverage describes the materials as part of the same campaign.

Use `confirmed` only for explicit links. Use `likely` or `tentative` for inferred links.

## Campaign / Message Labels

Create short descriptive labels to organize collection, such as:

```text
New Year family reunion
Urban running community
Anti-anxiety self-care
Premium craft origin story
Limited-edition collaboration
```

Do not treat these labels as final insights or creative concepts. They are routing labels for downstream summary.

## Channel Completeness

For each campaign/message, report which channels were found:

- Video
- Offline activation
- Marketing mechanism
- PR/media coverage
- Visual materials
- Social leads
- Report/context

This helps identify whether collection is complete or skewed.

## Campaign / Message Repetition Check

Track whether the same campaign/message appears across:

- Multiple materials in one campaign
- Multiple campaigns by one brand
- Multiple competitor brands
- Multiple platforms

Add notes such as:

```text
The same "family reunion" message appears in video, PR, and social leads, but offline evidence was not found.
```

Keep wording factual and evidence-bound.
