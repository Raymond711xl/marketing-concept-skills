# Dossiers

This folder stores exported backstage dossiers for real projects.

Default project structure:

```text
dossiers/[project-slug]/backstage-dossier.md
```

A dossier is structured project memory, not a public-facing report. It should preserve the original brief, evidence preparation brief, evidence pool, cleaned evidence, frontstage content strips, Strategy Readiness Pack, insight map, strategy candidates, Idea Platform records, Concept records, decision log, and open questions.

The controller should maintain this file by default for real projects, while showing only the relevant index, content strips, evidence brief box, changes, or selected sections in chat. Print the full dossier only when the user asks to view it.
